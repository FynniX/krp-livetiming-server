# krp-livetiming-server

LiveTiming Server for [Kart Racing Pro](https://www.kartracing-pro.com/).

## Requirements

- None

## Installing

- Download executable for your OS + Config.json
- Put them in the same folder
- Configure ```config.json```
- Run the executable
