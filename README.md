# krp-livetiming-server

LiveTiming Server for [Kart Racing Pro](https://www.kartracing-pro.com/).

## Requirements

- [Node.js](https://nodejs.org/)

## Installing

- Download Release
- Extract Release
- Run ```npm install```
- Run ```npm start```
